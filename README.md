# ai_enhancement

A collection of reusable Claude Code skills, automation scripts, and QA tooling for hardening AI-assisted development workflows.

## What This Is

This repository provides drop-in skills for [Claude Code](https://docs.anthropic.com/en/docs/claude-code) that enhance development quality, automate repetitive QA tasks, and enforce project standards. Skills are designed to be project-agnostic — install them globally or per-project.

## Skills

| Skill | Invoke With | Description |
|-------|-------------|-------------|
| **qa-skill-hardening** | `/qa-skill-hardening [target]` | Multi-agent QA swarm that auto-discovers, tests, fixes, and documents project skills using a 9-step binary test framework |

## Installation

### Global (available in all projects)

Copy any skill directory to your global skills folder:

```bash
cp -r .claude/skills/qa-skill-hardening ~/.claude/skills/
```

### Per-Project (committed to repo)

Skills in `.claude/skills/` are automatically available when working in this repository.

## Project Structure

```
ai_enhancement/
  CLAUDE.md                              # Project rules for Claude Code
  README.md                              # This file
  CHANGELOG.md                           # Version history
  .claude/
    skills/
      qa-skill-hardening/
        SKILL.md                         # QA hardening swarm skill
  docs/
    SUMMARY.md                           # Condensed project summary
    ARCHITECTURE.md                      # Structure and design decisions
    CONTRIBUTING.md                      # How to add new skills
    skill_memory/                        # Generated by qa-skill-hardening
      <Skill_Name>_MEMORY.md            #   Per-skill test results
      REGRESSION_TESTS.md               #   Regression test registry
      GIT_COMMIT_LOG.md                 #   Hardening commit history
      SKILL_HARDENING_MASTER.md         #   Global status tracker
      FINAL_SKILL_HARDENING_REPORT.md   #   Final hardening report
```

## Quick Start

1. Clone this repo
2. Open a Claude Code session in your target project
3. Run `/qa-skill-hardening` to audit and harden all skills in that project
4. Review the generated report in `docs/FINAL_SKILL_HARDENING_REPORT.md`

## Requirements

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI, desktop app, or web app
- Git
